# ColorQuest
Color Quest is a vibrant puzzle-adventure where players journey through chromatic realms to restore fading hues. Solve color-based puzzles, mix pigments, and unlock ancient palettes while confronting hue-shifting foes. Collect rare prisms, upgrade your brush, and reveal the world’s hidden spectrum. Experience a luminous tale of creativity, courage!
ColorQuest
